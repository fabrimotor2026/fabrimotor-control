# Renovación visual FabriMotor

Cambios aplicados:

- Fondo claro industrial más limpio.
- Cabecera y menú lateral más corporativos.
- Mejor contraste de botones azules, verdes, rojos y degradados.
- Tabla de histórico más legible: cabecera oscura con texto blanco y filas claras.
- Campos de formulario más visibles con foco verde FabriMotor.
- Botón base actualizado en `components/ui/button.jsx`.

Instalación:

1. Sustituye tu carpeta `src` por esta carpeta.
2. Ejecuta:

```cmd
npm run dev
```

Si Vite ya estaba abierto, detén con `Ctrl + C` y arranca de nuevo.
